# My Resume Project

This folder contains a tailored resume project for an upper elementary math curriculum role.

## Files
- index.html: Resume page
- style.css: Screen and print styling
- cover-letter.html: Matching cover letter for the same application
- one-page.html: Tighter one-page resume version
- resume-plain-text.txt: Plain-text version for online application forms
- photo-crop-guide.md: Exact instructions for saving and cropping the headshot

## Notes
- The resume text has been updated using the details visible in the provided resume image.
- To show the actual headshot, add a cropped image file named profile-photo.jpg to this folder.
- I could not enhance or sharpen the real photo because the attached image file itself is not available as a standalone editable file in the workspace.
- The portrait styling has been updated to be larger and softer with more flattering crop positioning.
