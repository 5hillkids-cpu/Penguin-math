# Photo Crop Guide

Use this guide when saving the attached portrait into the resume folder.

## Save Location
Save the image here:

- c:\Users\parki\Documents\My_AI_Course\my resume\profile-photo.jpg

## Best Crop
- Use a vertical portrait crop.
- Keep the face centered.
- Keep the crop from just above the hairline to around mid-chest.
- Leave a little space above the head so the top of the hair is not cramped.
- Keep both shoulders visible if possible.

## Recommended Image Setup
- File name: profile-photo.jpg
- Format: JPG
- Good target size: around 600 x 750 pixels or larger
- Avoid overly tight crops that cut into the chin or hair

## Resume Display Behavior
- The main resume uses a larger soft-corner portrait frame.
- The one-page resume uses a slightly smaller version of the same portrait style.
- The CSS is already set to crop higher on the image so the face stays prominent.

## Practical Steps
1. Open the attached photo in any image editor.
2. Crop it to a portrait orientation.
3. Save it as profile-photo.jpg.
4. Put it in the my resume folder.
5. Reopen index.html or one-page.html in the browser.
