# Georgia Mechanical Handoff Package

This folder organizes the Georgia Mechanical training rollout assets by function so a facilitator, supervisor, LMS administrator, or training manager can find the right materials quickly.

## Navigation
- index.html provides a browser-friendly landing page for this handoff package.

## Sections
- 01-program-admin: Program-level rollout, LMS, branding, and implementation materials
- 02-module-packages: Direct paths to each module package and delivery assets
- 03-assessment-and-supervision: Final exam, capstone, signoff, and coaching materials
- 04-print-and-brand-assets: Print-ready flyers, certificate, and brand collateral
