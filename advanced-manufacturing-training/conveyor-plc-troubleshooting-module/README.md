# Conveyor PLC Troubleshooting Module Package

This folder contains a first-pass advanced manufacturing training package focused on conveyor electrical and PLC troubleshooting.

## Included Files
- 01-module-overview.md: Audience, objectives, safety, competencies, and assessment approach
- 02-instructor-manual.md: Facilitation guidance, preparation steps, coaching prompts, and answer expectations
- 03-detailed-lesson-plan.md: Timed instructional sequence with checks for understanding
- 04-learner-workbook.md: Guided notes, data tables, practice questions, and exit ticket
- 05-virtual-lab.md: Scenario-based troubleshooting exercise with answer key
- 06-rise-storyboard-outline.md: Screen-by-screen interactive module outline for Rise or Storyline
- 07-rise-production-script.md: Block-by-block Rise authoring script
- 08-assessment-bank.md: Knowledge checks, performance task, answer key, and rubric
- 09-facilitator-slide-deck.md: Slide outline for instructor delivery
- 10-presenter-notes.md: Presenter prompts and debrief guidance
- 11-storyline-scene-script.md: Scene-by-scene Storyline interaction script
- 12-georgia-mechanical-slide-script.md: Fully written Georgia Mechanical branded slide script
- 13-georgia-mechanical-facilitator-guide.md: Speaker-ready Georgia Mechanical delivery guide with timing and activity flow
- 14-georgia-mechanical-course-flyer.md: One-page Georgia Mechanical course flyer
- 15-supervisor-observation-checklist.md: Georgia Mechanical field observation and coaching checklist

## Recommended Use
1. Adapt the overview and lesson plan to the actual conveyor hardware or simulator available.
2. Replace placeholder PLC tag names with site-specific tags.
3. Build the storyboard into Rise or Storyline as prework or reinforcement.
4. Use the virtual lab and workbook together during instructor-led delivery.
