# Assessment Bank

## Module Title
Conveyor System Electrical and PLC Troubleshooting

## Knowledge Check Questions

### Multiple Choice
1. Which condition most directly prevents the PLC from issuing a conveyor run command in the scenario?
   - A. Overload healthy input true
   - B. Jam clear photoeye input false
   - C. Start command bit true
   - D. HMI line status ready
   - Correct answer: B

2. If the PLC run output is true but the motor starter coil does not energize, the next checks should focus on:
   - A. Operator training only
   - B. Input devices only
   - C. Output circuit and field power path
   - D. Jam cleanup procedure only
   - Correct answer: C

3. What is the best first step after hearing “the conveyor will not restart”?
   - A. Replace the photoeye
   - B. Reset the overload several times
   - C. Confirm the symptom and review sequence of operation
   - D. Download a new PLC program
   - Correct answer: C

4. Which device most directly confirms the product path is clear?
   - A. Motor overload relay
   - B. Jam clear photoeye
   - C. Main disconnect
   - D. Start pushbutton
   - Correct answer: B

## Short Answer

## Performance Task
Given a no-restart conveyor fault, the learner must:
1. State the symptom accurately.
2. Identify the blocked permissive.
3. Name the likely root cause category.
4. Recommend corrective action.
5. List verification steps after repair.

## Answer Key Summary
- Blocked permissive: Jam clear photoeye
- Root cause category: Instrumentation with mechanical disturbance
- Corrective action: Realign the photoeye, verify PLC input state, and restart safely
