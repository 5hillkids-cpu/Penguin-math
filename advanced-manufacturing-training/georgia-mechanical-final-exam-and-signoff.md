# Georgia Mechanical Final Exam and Supervisor Signoff

## Exam Title
Georgia Mechanical Technical Troubleshooting Academy Final Exam

## Purpose
This final exam measures whether the learner can apply Georgia Mechanical troubleshooting standards across conveyor, HVAC, and chiller systems using evidence, sequence-of-operation reasoning, and safe corrective-action planning.

## Exam Format
- Part 1: 15-question written exam
- Part 2: Scenario-based oral explanation
- Part 3: Supervisor signoff rubric

## Part 1: Written Exam

### Multiple Choice
1. If a PLC start command is true but the run output remains false, the strongest next focus is:
   - A. Mechanical bearings only
   - B. Missing permissives or logic conditions
   - C. Immediate PLC replacement
   - D. Operator retraining only
   - Correct answer: B

2. In an HVAC no-fan-start fault, 24 VAC at the control transformer and 0 VAC at the relay coil most strongly suggest:
   - A. The motor has failed
   - B. The BAS is always wrong
   - C. An interruption in the upstream control path
   - D. The fan belt is loose
   - Correct answer: C

3. A chiller controller blocking compressor enable because it sees 33.0 F leaving-water temperature is:
   - A. Always incorrect logic
   - B. Potentially correct logic acting on bad input data
   - C. Proof the compressor is seized
   - D. A reason to bypass the safety
   - Correct answer: B

4. Which action best reflects Georgia Mechanical troubleshooting standards?
   - A. Reset first, diagnose second
   - B. Replace the most likely part to save time
   - C. Confirm the symptom, trace the sequence, then verify with data
   - D. Trust the HMI without field confirmation
   - Correct answer: C

5. Which device is most likely responsible for a blocked conveyor restart after a jam-clear event in the training scenario?
   - A. Main disconnect
   - B. Misaligned jam-clear photoeye
   - C. Compressor contactor
   - D. Pump proof switch
   - Correct answer: B

### Short Answer
6. Explain the difference between demand, permission, and command in a controlled system.
7. Why is an alarm message not always the final diagnosis?
8. Describe two reasons why restart verification matters after a repair.
9. What evidence would justify replacing a sensor instead of resetting the equipment?
10. Why should technicians distinguish symptoms from root causes?

### Scenario Response
11. Conveyor scenario: The start bit is true, run output is false, and jam-clear input is false. State the likely blocking condition and the next best diagnostic step.
12. HVAC scenario: BAS demand is true, control voltage exists, and freezestat is open. Explain why the contactor is not the first proven failure.
13. Chiller scenario: Handheld temperature does not match the controller temperature input. Explain what that contradiction means.
14. Give the six-step Georgia Mechanical troubleshooting process in order.
15. Describe what a supervisor should hear in a strong root-cause explanation.
