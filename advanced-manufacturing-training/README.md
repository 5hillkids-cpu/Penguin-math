# Advanced Manufacturing Training Packages

This folder contains modular technical training packages for industrial troubleshooting topics. Each package is organized for blended delivery and includes instructor-facing materials, learner-facing materials, a virtual lab, and a digital storyboard or Rise-ready script.

## Packages
- conveyor-plc-troubleshooting-module
- industrial-hvac-troubleshooting-module
- industrial-chiller-diagnostics-module

## Standard Deliverables
- Module overview
- Instructor manual
- Detailed lesson plan
- Learner workbook
- Virtual lab exercise
- Rise or Storyline storyboard outline
- Rise production script
- Assessment bank
- Facilitator slide deck
- Presenter notes
