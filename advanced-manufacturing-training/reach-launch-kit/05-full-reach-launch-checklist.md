# Georgia Mechanical Full Reach Launch Checklist

## Purpose
Use this checklist to launch all three Georgia Mechanical technical modules and the final assessment inside Reach 360 with consistent naming, completion rules, and learner experience.

## Course Set
1. Georgia Mechanical: Conveyor System Electrical and PLC Troubleshooting
2. Georgia Mechanical: Industrial HVAC Electrical and Control Troubleshooting
3. Georgia Mechanical: Industrial Chiller Control and Diagnostics
4. Georgia Mechanical Technical Troubleshooting Academy Final Assessment

## Naming Standard
- Course titles should begin with Georgia Mechanical:
- Learning path title should be Georgia Mechanical Technical Troubleshooting Academy
- Keep titles consistent across Rise, Reach, LMS listings, and certificates

## Pre-Launch Build Checklist
For each course, confirm:
- Course title is correct
- Course description is loaded
- All lesson titles are finalized
- Georgia Mechanical branding is applied
- Knowledge checks are functioning
- Completion rules are configured
- Mobile preview is acceptable
- Final publish package is exported

## Reach Setup Checklist
For each course in Reach:
1. Upload or connect the published Rise course
2. Confirm launch opens correctly
3. Set the thumbnail and course description
4. Add audience tags and metadata
5. Set availability dates if needed
6. Configure completion tracking
7. Test completion status with a pilot learner account

## Recommended Completion Rules

### Conveyor
- Completion type: View all lessons and complete final knowledge checks
- Passing target: 80 percent if scored

### HVAC
- Completion type: View all lessons and complete final knowledge checks
- Passing target: 80 percent if scored

### Chiller
- Completion type: View all lessons and complete final knowledge checks
- Passing target: 80 percent if scored

### Final Assessment
- Completion type: Pass final exam and complete supervisor signoff outside or alongside Reach workflow

## Learning Path Setup
1. Create learning path: Georgia Mechanical Technical Troubleshooting Academy
2. Add courses in this order:
   - Conveyor
   - HVAC
   - Chiller
   - Final Assessment
3. Require sequential completion if the academy is intended as a progression
4. Add due dates if this is assigned to a technician cohort

## Recommended Metadata
- Audience: Maintenance, HVAC, Mechatronics, Operations Support
- Keywords: troubleshooting, PLC, conveyor, HVAC, chiller, controls, maintenance
- Level: Intermediate
- Delivery mode: Self-paced or blended

## Pilot Launch Checklist
1. Enroll a small pilot group
2. Verify course launch on desktop and mobile
3. Confirm completion status updates correctly
4. Collect learner feedback on clarity and navigation
5. Resolve issues before broad assignment

## Supervisor Integration Checklist
- Supervisors receive the signoff rubric
- Observation checklists are assigned by module
- Coaching forms are available for remediation cases
- Final signoff workflow is communicated clearly

## Post-Launch Review
After the first cohort, review:
- completion rates
- quiz performance
- learner feedback
- supervisor signoff quality
- needed revisions to examples, screenshots, or wording
