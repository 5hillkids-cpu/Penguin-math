# Final Assessment Reach Setup Guide

## Purpose
Use this guide to configure the Georgia Mechanical final assessment and signoff workflow after the three technical modules are available in Reach.

## Assessment To Configure
Georgia Mechanical Technical Troubleshooting Academy Final Assessment

## Source Files To Keep Open
- ../georgia-mechanical-final-exam-and-signoff.md
- ../capstone-troubleshooting-assessment.md
- ../georgia-mechanical-supervisor-coaching-forms.md

## Recommended Final Assessment Structure
The final stage should include:

1. A written knowledge check or quiz
2. A capstone troubleshooting scenario review
3. A supervisor signoff step outside or alongside Reach

## Part 1: Create The Final Assessment Item In Reach
1. Open Reach 360.
2. Create a new learning item or assessment placeholder, depending on your Reach configuration.
3. Name it: Georgia Mechanical Technical Troubleshooting Academy Final Assessment.
4. Use this short description:
   Complete the final written assessment, scenario review, and supervisor signoff requirements for the Georgia Mechanical Technical Troubleshooting Academy.

## Part 2: Decide How Reach Will Handle The Assessment
Choose one of these models:

### Option A: Reach Tracks A Final Course
- Build the written exam in a supported quiz or linked course format.
- Use Reach only to track the completion of that exam.
- Handle supervisor signoff outside Reach with the signoff document.

### Option B: Reach Tracks A Completion Step
- Create a completion item in Reach called Final Assessment.
- Instruct learners to complete the written exam and supervisor signoff externally.
- Mark completion after supervisor verification.

### Option C: Blended Tracking Model
- Use Reach for the written assessment and capstone completion.
- Use supervisor signoff as a separate documented follow-up step.

## Part 3: Build The Written Assessment
1. Use the questions in ../georgia-mechanical-final-exam-and-signoff.md.
2. Create a 15-question assessment aligned to the final exam structure.
3. Keep course naming consistent with the academy naming standard.
4. Set a passing score aligned to your internal policy.
5. Add retake rules if needed.

## Part 4: Add Capstone Requirements
1. Reference ../capstone-troubleshooting-assessment.md.
2. Decide whether the capstone is:
   - facilitator graded
   - supervisor graded
   - discussion based
   - documented outside Reach
3. If Reach cannot host the capstone directly, use the final assessment item description to explain the offline requirement.

## Part 5: Add Supervisor Signoff Workflow
1. Use ../georgia-mechanical-final-exam-and-signoff.md for the signoff rubric.
2. Use the module supervisor observation checklists as supporting evidence.
3. Use ../georgia-mechanical-supervisor-coaching-forms.md when a learner does not meet standard on the first attempt.
4. Decide who records the final signoff status.

## Part 6: Recommended Completion Rule
Mark the final assessment complete only when:
1. The written exam is passed
2. The capstone requirement is completed
3. Supervisor signoff is documented

## Part 7: Add It To The Learning Path
1. Open the Georgia Mechanical Technical Troubleshooting Academy learning path.
2. Add the Final Assessment item after Conveyor, HVAC, and Chiller.
3. Require sequential completion if the full academy is structured as a progression.

## Part 8: Pilot Test The Workflow
1. Run one test learner through the three courses.
2. Confirm the final assessment appears in sequence.
3. Confirm completion logic works as expected.
4. Confirm the supervisor knows how to use the signoff and coaching forms.

## Practical Recommendation
If Reach is limited for graded practical evaluation, use Reach to track the written final assessment and use the supervisor signoff workflow outside Reach as the formal performance gate.
