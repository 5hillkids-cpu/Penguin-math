# Georgia Mechanical Rise Build Checklist

## Purpose
Use this checklist to convert the Georgia Mechanical training package into self-paced online courses built in Rise 360 and delivered through Reach 360.

## Before You Build
1. Choose the module you want to publish first.
2. Gather the module source files:
   - Rise production script
   - Assessment bank
   - Georgia Mechanical slide script
   - Course flyer or branding pack if visuals are needed
3. Decide whether the module will be:
   - fully self-paced
   - blended with instructor-led delivery
   - prework before a live session

## Rise Build Steps
1. Create a new Rise 360 course.
2. Name the course using the Georgia Mechanical branded title.
3. Add a course description based on the Georgia Mechanical LMS shell copy.
4. Create one lesson for each major lesson group in the Rise production script.
5. Add blocks in the same sequence shown in the script.
6. Paste on-screen text from the script into the matching Rise block types.
7. Add knowledge checks using the assessment bank.
8. Apply Georgia Mechanical branding, colors, terminology, and examples.
9. Add diagrams, screenshots, or plant-specific visuals where available.
10. Set lesson labels and navigation rules.
11. Define completion settings.
12. Preview on desktop and mobile.
13. Publish for LMS or Reach delivery.

## Recommended Rise Block Mapping
- Text on image: scenario introduction
- Statement block: key concept reinforcement
- Process block: sequence of operation
- Accordion: safeties, permissives, or interlocks
- Labeled graphic: signal path or system diagram
- Sorting activity: classify fault zones or symptoms
- Flashcards: troubleshooting prompts
- Table: scenario evidence
- Knowledge check: comprehension verification
- Continue block: lesson summary and transfer step

## Branding Checklist
- Use Georgia Mechanical course title
- Keep tone direct and technical
- Use site-specific tags or screenshots where possible
- Keep examples aligned to maintenance workflows
- Use high-contrast visuals and readable diagrams

## Reach Delivery Steps
1. Publish the Rise course in the required format for your Reach setup.
2. Upload or connect the course in Reach 360.
3. Set learner audience and enrollment rules.
4. Add due dates if required.
5. Group the course into a Georgia Mechanical learning path or academy.
6. Test learner launch and completion tracking.

## Quality Check Before Launch
- All lesson titles match the intended module flow.
- Knowledge checks are functioning.
- Completion rules are set correctly.
- Links, images, and interactions work on mobile and desktop.
- Georgia Mechanical terminology appears consistently.

## Launch Recommendation
Start with the conveyor module first because it is the simplest sequence-driven example and creates a repeatable pattern for HVAC and chiller builds.
