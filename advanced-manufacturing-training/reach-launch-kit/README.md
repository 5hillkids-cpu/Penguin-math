# Reach Launch Kit

This folder contains the practical documents needed to turn the Georgia Mechanical curriculum package into Reach-delivered online courses.

## Included Files
- 01-rise-build-checklist.md: Step-by-step checklist for building the courses in Rise 360
- 02-conveyor-rise-copy-paste-outline.md: Copy-paste-ready lesson outline for the conveyor module
- 03-hvac-rise-copy-paste-outline.md: Copy-paste-ready lesson outline for the HVAC module
- 04-chiller-rise-copy-paste-outline.md: Copy-paste-ready lesson outline for the chiller module
- 05-full-reach-launch-checklist.md: End-to-end checklist for launching all courses in Reach
- 03-reach-course-map.md: Recommended Reach course structure and launch order
- 06-conveyor-rise-click-by-click-build-guide.md: Literal build order for constructing the first Rise course screen by screen
- 07-hvac-rise-click-by-click-build-guide.md: Literal build order for constructing the HVAC Rise course
- 08-chiller-rise-click-by-click-build-guide.md: Literal build order for constructing the chiller Rise course
- 09-final-assessment-reach-setup-guide.md: Practical setup guide for the final assessment and supervisor signoff workflow

## Recommended Starting Order
1. Build the conveyor module in Rise
2. Build HVAC and chiller using the same outline pattern
3. Pilot the conveyor module in Reach
4. Launch HVAC and chiller after pilot validation
5. Add the final assessment and supervisor signoff workflow
