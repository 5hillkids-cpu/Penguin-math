# Learner Workbook

## Lesson Title
Industrial HVAC Electrical and Control Troubleshooting

## Learning Goals
Use sequence-of-operation logic, safety-circuit checks, and electrical measurements to isolate why an HVAC unit fails to start or trips offline.

## Key Terms Table
| Term | Meaning in this system |
| --- | --- |
| Freezestat | Safety device that opens when low temperature threatens coil freeze-up |
| Control transformer | Device supplying control voltage for relays and controls |
| Fan contactor | Electrically operated switch that energizes the fan motor circuit |
| Proof switch | Device confirming airflow or device movement |
| BAS command | Building automation request for operation |

## Guided Notes
1. A BAS run command does not guarantee the fan will start because all ________ must still be satisfied.
2. If control voltage is missing at the relay coil, the problem is upstream in the ________ path.
3. A safety device opening is often a ________ clue, not a reason to guess.

## Troubleshooting Log
1. Record the complaint.
2. Identify the first three checks.
3. Note where the control path stopped.
4. State the root cause.
5. Record the corrective action and verification steps.

## Practice Questions
1. Why is a fan contactor not the first part to blame in a no-start fault?
2. How would low control voltage create misleading symptoms?
3. What evidence proves the BAS signal is not the limiting issue?

## Exit Ticket
Explain how you would prove that an open safety string, not a failed motor, caused the unit not to start.
