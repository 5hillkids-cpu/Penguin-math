# Rise or Storyline Storyboard Outline

## Module Title
Industrial HVAC Electrical and Control Troubleshooting

## Screen 1
- Title: The Unit Is Called, But Nothing Starts
- On-screen text: The BAS shows occupied mode. The building needs airflow. The fan stays off. Diagnose the fault using sequence logic.
- Visual direction: Rooftop unit image with BAS overlay
- Interaction type: Intro scenario

## Screen 2
- Title: Sequence of Operation
- On-screen text: Follow the path from BAS demand to fan motor start.
- Visual direction: Animated control path
- Interaction type: Click-through process

## Screen 3
- Title: Safety String
- On-screen text: One open safety can stop the entire start sequence.
- Visual direction: Series safety loop with hotspots
- Interaction type: Hotspot reveal

## Screen 4
- Title: Trace the Voltage Path
- On-screen text: Find where the control signal stops.
- Visual direction: Schematic with checkpoints
- Interaction type: Guided troubleshooting

## Screen 5
- Title: Make the Diagnosis
- On-screen text: Use the evidence to choose the most likely root cause.
- Visual direction: Multiple-choice scenario cards
- Interaction type: Knowledge check

## Screen 6
- Title: Verify and Restart
- On-screen text: Correct the issue and prove stable operation.
- Visual direction: Restart checklist
- Interaction type: Ordered steps
